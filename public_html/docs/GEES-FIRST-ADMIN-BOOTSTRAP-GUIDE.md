# First Admin Bootstrap Guide

1. Create a normal user through Supabase Auth or the website signup page.
2. Open Supabase SQL Editor.
3. Run `public_html/supabase/sql/phase10e_first_admin_promote_by_email.sql` after replacing the email.
4. Login from `/portal/auth/admin-login.html`.
5. Open `/portal/admin/approvals.html`.

Demo admin cannot read real Supabase approval rows.
